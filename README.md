# gyapp

## Install (Under development)
```bash
npm link
```


## Usage
```bash
# Take screenshot of web site and upload to Gyazo
gyapp https://google.com/

# Get images of specific Twitter user and upload to Gyazo
gyapp-twitter yuiseki

```
